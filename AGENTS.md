# AGENTS.md

This is a paranoid-text-spacing project, so its own docs must not drift.
When editing CJK-mixed prose here (README.zh-TW.md, SKILL.md examples,
commit messages): use full-width punctuation in Chinese（，。：；（）「」）,
not half-width, and confirm pangu.js `spacingText()` leaves the prose
unchanged before committing. Rules live in skills/pangu/SKILL.md.
